明天去义乌

发表于 2020-07-13 分类于 小说

游荡在黑夜里的人突然发现自己置身于彻底的光明之中，忘记了这一切只是由于眼皮被轻轻睁开。四周的一切东西，棱角分明的墙壁两端相接搭成空间，有着温柔褶皱的衣服安逸地垂下袖口，窗外的蓝天带着一抹淡淡的灰色，填满了眼前的每一个角落。虚无并不存在。另一侧的窗户那里，微黄的阳光被框架切割成几块长长的矩形，洒在有几块污迹的白色地板上。也许有鸟儿在窗外鸣叫，只是他没有听到。在别的地方太阳正在缓缓地升起，先是把几片薄薄的云彩染成绚烂的金黄色，自己再从群山背后或者大海的另一端一跃而起。又或者它一身金黄，却只是静静地挂在天边，连射出的光线都是那么柔和，好像世间为它所照亮的万物，它们逐渐浮现的轮廓都与它毫无关系。

梦中的景象迅速地在他脑海里消失着，有一张女人的脸，是他所认识的人，但是谁？他想不起来。还有一些别的人，在梦中他们想麻雀一样聚在一起唧唧喳喳地说话，随后又像怕光的吸血鬼那样，在这个光亮的世界里一哄而散，只剩下他一个人留在牛奶般的空白中，绞尽脑汁地回想到底发生了什么。同时觉得自己在这场努力中越陷越深，沉入无尽的虚无，挥舞着的双手抓不到任何东西。棱角分明的房间取代了那张女人的脸，三条互成九十度的直线是现实的最佳代表。义乌，是的，明天要去义乌。这条讯息写在墙壁上，写在袖口上，当他望向高楼背后的蓝天时，他知道自己明天就要去义乌了。

于是他开始在空荡荡的房间里自言自语起来，讲述一场尚未发生的旅程。这里的一切都是他的镜子，帽子知道他头颅的轮廓，杯子记得他握杯时的姿势与力度，床铺则记录了他每天早上压出的那道弯曲的痕迹。“是的，我明天就要去义乌了”他说，“那里可能有未来，也可能有未曾谋面的爱人。为了去一次义乌，我们必须要上网买票，必须在六点起床，必须搭乘正确的地铁线路到达火车站，必须把票递给检票员。一路上我能看到很多东西，也可能什么也不加注意。无数的脸孔从我眼前飘过让我有可能害怕生活，重新躲进自己温暖安全的小窝，也可能使我对生活轻蔑地一笑。让我们在想象中来到义乌吧，让一切期待，恐惧，失望，骄傲在想象中先行上演。不用担心虚构不出小商品城的位置和它的几个出入口，它的门把手是黄是红都没什么紧要，虚构的故事不会出错且会给人以教益。”

A 一个声音说，鞋子上映出一对阖动不已的嘴唇“我想去义乌，我一直就想去义乌。那里有最大的小商品批发市场，有最忙乱的生活，公交车载着满满当当的乘客，从一张纸币开到另一张纸币，钞票的味道弥漫在每一个角落，而我想品尝吞噬这种气味。我能见到商铺紧紧地挨在一起，陈列的货物从狭小的空间里流溢出来，为了在里面行走我们不得不跨过这个，绕过那个。最终，我们走进了一家店铺，让自己进一步被商品包围，摆在货架上的，地上的，挂在墙上的，拿在老板与顾客手里的，还有他们嘴里说着的。老板会看我一眼，再无精打采地垂下目光，或者继续将目光与期望捆绑，直直地投向外面路过并东张西望的人群。我什么也不会说，什么也不会拿起来，不会问老板这个什么价钱，可不可以便宜一点。面对着它们，我总会想起二十年前，我在这样的一家批发商铺想买一毛钱果冻时的情景，老板说一毛钱我们不卖。那五毛钱你们卖不卖？老板接过我的五毛钱，从篮子里随便抓给了我一把。我边往回跑边把它们塞进口袋，接着告诉妈妈，我用你给的钱买了果冻。还有，把果冻盖子揭开时手一定要稳，不然里面的水会撒出来，那就喝不到了。从那时候起，我走路变得很稳，手也变得很稳，当你打破生活的盖子时，各种危险都会洒出来。我头上的天空到处悬挂着隐形的利剑，我的脚下是一层薄冰，可能有野兽躲藏在右手边的灌木丛之中伺机而动，因此，我只能小心行事。

“我只是静静地看，同时也承受它们来自四面八方的目光，有的看到了我的鞋子，有的看到了我掖在耳后的头发，有的则注意到了我粗大的指节。它们有的是时间来彼此交流，最终拼出一副完整的我的肖像，随后把我彻底遗忘。而我却始终能听到时钟在滴答响个不停，一分钟逝去了，我仿佛看到了凶器隐约的寒光。有陷阱在前面等着我，而我则被时间一步一步地向前推着走。还是它们获胜了，我在它们的目光下节节败退，在它们给我的肖像面前无地自容。许多句话在我脑子里酝酿出来，几乎已经到了嘴边，我想显得有经验，老道，可能还有些钱，这样我就对它们而言具有意义。我想把它们拿在手里，翻来覆去地观察摩挲，最后在失望地放回去，微微地摇下头。但这些举动在它们面前都不过是儿戏。尽管我伸手去拿某件货物（老板在我背后微微一笑，并没有出声阻拦），把它牢牢地抓在手里，可它闭上了眼睛，蜷缩起身子，把我当成一个绝对的外人阻挡在外面。一个部件紧扣另一个，螺丝拧得紧紧的，胶水还能经受多年时光的侵蚀，经线与纬线密不可分甚至找不到起始点在哪里。我只能把它原样放回去，摆在同样的位置，假装一切都没有发生。我转身离去时听到了它们正在背后窃窃私语。”

B “啊，我记起了那个女人，还有她和别人一前一后走在沙滩时的模样。永不停歇的海浪在他们身后仿佛静止下来，海浪冲上沙滩时泛起的白色泡沫仿佛始终就在那里，那一排排昂起的浪头成了安静的观众，被困在固定的座位上，和我一起看她的裙角如何微微摆动。她在身后留下一个个脚印，接着其他的脚印也纷纷跟了上来，她回头一笑，他们都笑了。我讨厌自己只是一个观众，更讨厌她离去后空荡荡的舞台。海浪无尽地漫过沙滩，接着再将一个个泡沫戳破，让它们变成不可见的小河流回大海。海浪抚平了她和其他人的脚印，一切都像未曾发生。海浪变回了海浪，我也意识到这只是一个梦。可她曾经真的走过这片海滩，和其他人一起，而我则躲在一棵树的背后偷窥。一头野兽在舔舐自己的伤口，它喜欢血的味道，哪怕那是它自己的血。她走了，所以我也醒了。

“所以就当作那是义乌的海滩吧，（可义乌没有海滩），就把明天的旅途当作是寻找梦中爱人的努力吧。’她不是你的爱人，她和别人一起走，一起笑‘从路过的每一处风景，遇见的每一张面孔上，我都能找到它们和她之间的关系。她像春季的河流一样安静，像刚刚长满绿叶的树木一样可爱，当一只飞鸟飞过天边时，总能提醒我她走了之后那片沙滩是多么的空旷。也许我的对面位子上就坐着一位年轻的女人，可我不会看她。我靠着窗户，手里拿着一本书，一路上只是望着外面。可我无时无刻不在感觉到她就在我对面。因为我没有仔细地看她，她就成了那条河，那棵树和那只飞过天边的鸟儿。她也是一片云，彩色且形态多变，即使是在没有风的车厢里。”

正午时分，被太阳照亮的一切不再显得暖洋洋，完全没有了原来懒散的模样。到处都是光明，是忙碌，新绿的树叶大口大口地吞食着阳光，女人们举起阳伞抵抗着阳光。建筑物失去了清晨时的残存的几分模糊与朦胧，在此刻的阳光下，它们好像变成了会闪闪发光的金属。天空也失去了那抹优雅的灰色，光天化日下，人们四处寻找着一个屋顶，从一个阴影跳跃着走向另一个阴影。

C “仅仅是去一个地方而已，我们不应该思虑的那么多。还是想想我们将要看到和听到的一切吧，明天的生活将会和今天的大不相同。我厌倦了无尽的阅读，厌倦了总是在重复的地方打转，也许是因为这个原因我才提议说’嗨，让我们去义乌吧‘。可能不是因为我对义乌有着什么特别的兴趣，而是因为义乌在别处，可以让我体验不同寻常的生活，仅此而已。啊，生活，我对你是多么热爱，可你却总是躲藏在别处。为了品尝不一样的生活，我不得不经历上述思虑和麻烦，但这一切终究是值得的。此刻虽然被困在这把椅子上，我已经开始为了新的经验做准备了。想象就是我为了明天的生活的大餐所品尝的开胃酒。只要能逃开这把椅子，那么一切都是值得的。清晨的地铁上会有几个人和我坐在一起呢？又有几个人在我抵达时留在我身边？我会在晨光微曦时走过空当无人的街道，两旁睡眼惺忪的水杉仅仅归我一人所有。在转身走进地铁站时，我会回头看一眼马路上沉默的汽车，默默地和他们说声再见。在地铁站等车时，暗色的玻璃将映出我的样子，被模糊的细节让我感觉很好。接着我置身于众人之间，身边漂浮着一些挂着眼屎的面孔，不时传来几声像是某个回音的哈欠。不同的面孔总是让我着迷，我总是暗暗地观察并在心里加以解释。’这是个上班族，外地人，可能上班的地方离他的出租屋很远所以他才起得这么早，他的被子有没有叠呢？’ ‘而那个像是入定一般的老者则是本地人，但过得很是凄苦，脸上的每一道皱纹仿佛都在像他一般对世界闭起了眼睛，认定生活能提供的不过是无尽的苦难与争吵’ ‘戴着耳机并低头玩手机的年轻女人总是让我观察的兴趣十足，。当耳机塞进耳朵里，眼前也只有手机屏幕时，现实世界消失了，她现在虽然和我们一道被关在车厢里，实际上却置身于另一个世界。正是这个世界让我着迷，那里有什么？随她去吧，我不愿去想，因为很可能和她的这个世界一样贫瘠。在某一站她将下车，站起身时将手机紧紧握在手里，她们几乎从来都不会坐过站。’

“仅仅在一节车厢我就能有这麽多的收获，那么就不难想象义乌能提供给我的有多少了。但还没到站我已经感到了厌倦，就像河底的卵石厌倦了河水的冲刷那样。确实，在义乌我能见到很多新奇的东西，像是庞然大物的小商品城，推拉着小货车的苦力在过道里四下奔走，某一家店铺坐着个漂亮的老板娘。但这就是所谓的‘生活’吗？假如生活真的只是一些经验的堆积，那只能说到了义乌的我和坐在椅子上的我一样没有生活。我确实地知道，等我晚上坐最后一班地铁回来时，我将会多么厌恶身边的人。厌恶他们的姿态，厌恶他们的面孔，厌恶那声无名的哈欠。甚至厌恶起了自己，因为自己竟然和这样的人同路，不管他们身上依然带着一整天的谎言，一整天的算计和卑躬屈膝加在一起的那种臭味。为了对周遭不闻不问，我也将闭上眼睛，我将看到一盘馒头摆在一张铺着塑料桌布的桌子上。如果睁开眼，地铁车厢的玻璃上就会浮现一张疲惫且充满厌恶的脸，这张脸和其他的并没有什么两样。”

D “我是一个梦想家，在一把椅子上做梦和在义乌做梦对我来说没什么两样，甚至就连义乌和它的一切都不过是我的想象。在开往义乌的火车上我的目光越过重重树叶，借助最微小的通道一直投射到山的那一边。有一颗小树孤零零地站在路边，有一张纸头平铺在马路上，上面有着几层脚印。如果你们想听我在义乌见到了什么，我马上就能编出一个故事来，没有错处供你们指摘。我可以完整地向你们复述一男一女之间关于昨天吃得饭店如何的对话，仔细描绘出那个女人扎在脑后的松散马尾。关于那张纸头和那棵树我也能说出好多，甚至会编造出一个男人，他先是靠在小树身上抽完了一支烟。起身的时候小树从微微倾斜变成竖直，嫩绿的叶子集体摇摆了几下，他转身继续走，毫不留情地踏过那张无辜的白纸。背对着他，几辆尾部挂着红灯的小汽车迅速在他身后小时。宽阔的马路一直向前延伸，路灯的队列像是没有尽头，马路逐渐上升，最终抵达天空。所有这一切都与我毫不相干，尽管我一会儿站在树前面观察一个男人，又低头研究纸上的脚印，接着站在斑马线上沿着马路极目远眺。如果你想知道地平线的另一边有什么，我可以马上原地升空，让灯光在脚下闪烁。我无处不在同时又全然不存在。但这一切就像是发生在一个大大的肥皂泡泡里。在我与世界之间一缕阳光投下，让这些想象中的关系变得生动又彩，但并不是真实，只要试图伸出手指触碰，一切都烟消云散。”

“悔恨始终就缠着我，它们会不会跟我一起去义乌呢？换言之，在义乌的什么地方，街灯下，酒吧吧台旁，一棵法桐下的长凳上，我会不会也会想起那些失去的机会和再也没能碰面的爱人。那天要是我那么回答他的面试问题就好了，要是我能在她转身离去时出言挽留，那么生活将大有不同，甚至我就失去了去义乌的理由。如果，如果，这个词给生活又添加了一层不幸。还有一层是由希望添加的，也许我去义乌要看的不仅仅是那些玲琅满目的商品，而是希望。希望的另一个名字叫也许，也许我会遇见一个女人，与她相爱，在若干年后共同牵着一个小孩的手走过铺上一层秋叶的义乌街头。也许我会在汹涌的人流中，就在哪个缝隙里发现赚钱的机会，再勇敢地抓住。生活也许会和现在的大不相同。”

“还没成为现实却早已变成了话语，所以说话的人身在何处？当孔子说’逝者如斯夫，不舍昼夜‘时，他正站在一条河的旁边。诗经里说’我心匪石，不可转也‘时，说话的人面前肯定有颗石头。而当梦想者谈论他的梦想时，他面对着一条流淌在苍白河床上的河流，落花与落叶被河水裹挟着流向远方。一时间，梦想者盯着这些五彩的花，叶可能还有大段的朽木，看得入迷。当他开始谈论起它们时，它们已消失不见。于是他说，这是最好的结局，有时候梦想成真就是为了毁掉那些做梦的人。有一次我从一个噩梦里醒来，摸到了脖梗后面的冷汗。我起身上厕所，昏暗的走廊让我很是犹豫担心，别人挂在两旁的衣裤不禁被我想象成是吊死的尸体，有手有腿在微微晃动。可另一方面我又很清楚那是衣服，仅仅是衣服，亦不会有鬼突然冲出来掐我的脖子。梦里的事情也不会成真，而是常常相反，所以我并不会在买袜子的时候遇到变态杀人狂魔。而那两把摆在桌子上的美工刀，一把带鞘，另一把只是刀片也不会向我提出问题；选我还是选他？和之前的很多梦一样，你都会慢慢把它们忘记掉。”

“有一些是不会忘记的，因为想到它的次数太多，反而把它和真实发生的事情混为一谈。确实有一次我和她在雨天一起走过高中门口那条泥泞不堪的煤渣路，一起经过一个倒在路边的乞丐。我几乎记得一切细节，比如她脚上的鞋子，乞丐乱糟糟的头发，还有漂在旁边池塘绿藻中间的白色塑料袋。真实的程度和那天我见到她和别人一起走，一起笑不相上下。”

“我们还在谈论义乌吗？听起来和义乌已经没了什么关系。”
“我不知道”
“我也不知道”
“那明天还要去义乌吗？”
“我不知道”
“我无所谓”
“只要能让我继续做梦就好。”
“我只希望明天永远不要到来。那么我们就可以将今天来来回回经历无数遍，一层层地扒开它的皮。在任何时候，当别人问起，我们都可以回答说，明天去义乌，或者明天不去义乌。”
