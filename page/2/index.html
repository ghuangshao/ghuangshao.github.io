腌笃鲜小记

发表于 2020-07-20 分类于 美食专栏

对于不甚了解苏州话的朋友，“腌笃鲜”听上去像是个名词。不过如果用苏州话来解释，它其实算是一句话。“腌”指的是咸肉和火腿，“笃”意思是“小火慢炖”，鲜则代指鲜肉和新上市的竹笋。所以“腌笃鲜”翻译成普通话就是，用咸肉和火腿小火慢炖鲜肉和竹笋。

苏菜时令与不时不食

发表于 2020-07-20 分类于 美食专栏

稍微了解我国饮食文化的人，大概都听说过“不时不食”。此语出自孔子的《论语·乡党第十》，意思是说吃东西要应时令、按季节，到什么时候吃什么东西。虽然说随着时代的变迁以及食物保存技术的进步，食材的“时令性”在现代已经没有那么明显了。但是在有的菜系中，依然保留着菜肴“随着时令的变化而改变”的传统。就“不时不食”而言，苏州菜可算得上其中的典范。

2019 林忆莲

发表于 2020-07-14 分类于 评论

时隔六年，林忆莲这次给我们送来一份厚礼，新专辑《0》从各方面来说都颇为惊艳。这首先应该归功于专辑背后强大的制作班底，尤其是她和著名音乐人常石磊的再次合作。两人此前合作的专辑《盖亚》让他们分别获得了第24届金曲奖最佳女歌手以及最佳编曲人奖，林忆莲的唱功以及常石磊的编曲能力助力《盖亚》获得最佳国语专辑奖。这一次，常石磊同样负责专辑《0》的编曲、和声编写与剪辑，同时也为几首单曲《太阳系》、《归零》、《幻觉动物》作词作曲。

如果说上一张专辑《盖亚》所探讨的是自我与世界的相处与接纳，而《0》更多地是在诉说女性的意识与感受体验。编曲风格上，整张专辑也是颇为多样。开篇的《breath in breath out》用低沉舒缓的钢琴声为人声伴奏，像是女性在午夜时分的自我对白，沉思应当如何生活每一天。《太阳系》的编曲风格更加迷幻，歌词也似乎失去了具体的所指，虚虚实实之间营造出迷幻的意境，让听者仿佛乘着歌声漂浮在太阳系之中。而到了《沙文》，风格则变成了电摇滚，节奏更加热烈，编曲富有层次感，歌词也从飘渺变得沉重起来。

《沙文》作为整张专辑的主打歌，所讲述的是一场女性意识的觉醒，由Sandy集结一批男性音乐人制作而成，其中包括男友恭硕良，作词者吴青峰，编曲常石磊以及和声五月天阿信。林忆莲先是以控诉的语气追问男性女性之间的暴力是“基因的遥控还是控制的诱惑”。在这种男女关系中，女性往往是作为被压迫的一方，如她所唱的“隐形暴力”、“恐惧下的沉默”、“冷漠与伤痕”等等。女人如花，却在凋谢之后往死里推，尊严被任意践踏。直到女性的意识开始觉醒，灵魂生根，才看出女性的被压制只是一场虚假的泡沫。

单曲《归零》可以看作是对《沙文》的续写，对Sandy来说，归零意味着重生。常石磊如是评价她：Sandy唯一没变的就是她始终在变化。不断变化意味着始终在归零与重生，这种喜欢变化的性格，让她从不惧怕未知，而是把未知当成是唯一的期待。这句歌词我尤其喜欢“既然 未知 是唯一的期待/索性 将平淡劈成干柴”。与歌词相应，配乐的感觉更加热烈乐观。为过去而悔恨并不值得，明亮紧随着黑暗到来，值得期待的只有未来。林忆莲的这首歌非常富有感染力，是我本张专辑的最爱。

热情过后，《0》重新回归从容，气息舒缓。《幻觉动物》词曲均由常石磊包办，曲调轻灵，歌词优美，与林忆莲富有磁性的嗓音搭配相宜。她的低吟浅唱像是在诉说女性情感的细腻，但又不全是，而是与某种更高渺也更深入内心的东西相连，使人沉醉其中，如闻天上仙乐，如见海市蜃楼。

《一呼一吸》听上去饱含禅意，Sandy像故事里禅师一般拿云朵、石头、黑豹和数目举例，其目的是为了点醒听者和自己——最美的音乐是生命原始的内在之音，需要用心去倾听。而生命就在这简单的一呼一吸之间，有起有伏，放掉才见充实，周而复始。而整张专辑的命名也体现了这层意思，《一呼一吸》与开篇的《breath in …breath out》首尾呼应，画下一道完整的圆，涵盖了林忆莲六年之中对生命的思考与感悟，其中有午夜的沉思，也有女性意识的觉醒，更有觉醒后向未来与未知的出发。我们很幸运，有这张专辑陪着我们走进2019，新的一年，思考我们自己的归零与重生。
