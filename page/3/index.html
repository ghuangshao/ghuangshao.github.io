老李的故事

发表于 2020-07-14 分类于 小说

去嫂子门上讨房子的路上，老李知道自己所走的是一条毁灭之路，等到了路的尽头，自己历经几十年风雨残存下来的那点尊严与廉耻将一点不剩。他虽不情愿可在妻儿的催逼下仍旧一步一步的走着，换乘倒车，灰蒙蒙的天空让他对路过的风景视而不见，一切都是灰色的，灰色是衰老的颜色,老李的头发也是灰色的。收回投向车窗外的目光,老李叹了口气，低下头打起了瞌睡。公交车走走停停，车上永远是人挤人，满满的全都是，人人都厌恶彼此。老李坐在最后一排，心思没在车上，也没在嫂子的房子上，我也不知道是在哪儿。大概他那个年纪的人的心思是很难猜透的，也许他脑子一片空白好腾出地方去对付并不好惹的嫂子，也许他在想三十年前去海南岛是见到的高大浓密的菠萝蜜树，也许是在对着前面秃顶男人的后脑勺发呆，油光水滑的后脑勺 模模糊糊的反映出老李中风后有些歪斜的嘴巴。老李下了公交之后又坐了地铁，照样是人挤人和不知在哪里的老人心思，他所走的路线和上次他媳妇，也就是老张走的路线一模一样，虽然两个人彼此没有知会。四十年的共同生活总会在两个个体之间建立起某种默契。

孩童公墓

发表于 2020-07-14 分类于 小说

我一直没能弄明白为什么这座看起来漂亮可爱的公墓每到星期天的时候总是不见人影，而附近几座要严肃的多也丑的多的墓地却总是有络绎不绝的游人，打着雨伞或者阳伞。在这里—-我的可爱公墓，偶尔也会见到除我之外的其他人光顾，边走边左顾右盼，好奇地打量墓碑上的铭文，然后消失在某一处拐角，之后再也没出现过。而且他们看起来要么和我一样是刚来这个城市工作的年轻人，要么是匆匆而过的游客，本地人是不会来这里的。由于跟他们不熟，再加上他们每个人都带着一副冷冰冰难以接近的神情，以后大概也没机会向当地人表达我的疑问。管他呢，反正我乐得独享这里的美景和在世间越来越难寻觅的安静。我是个喜欢安静的人同时也是活泼的人。一般我都是星期天去那里待上一下午或者一上午，这要看我能不能在十点之前从床上爬起来。去的时候我会先找家便宜餐馆饱餐一顿，再买上一瓶酒或是一包香烟，把酒瓶夹在腋下晃晃悠悠地向公墓走去。公墓没有名字，位置也有些偏僻但离我租住的地方并不远，一次偶然的机会我才得其门而入。那是一个突降暴雨的下午，具体我就不说了，反正我那天在墓地待了一下午。躲在守墓人小屋的屋檐下，暴雨把我和世界隔开，到处是白茫茫的，大粒的雨滴像早上挤地铁上班的人一样匆匆忙忙地从天外赶来，接着啪啪地落在地上不见了。晴天的时候四周高大浓密的松树林也能把世界隔在外面，并在里面落下一片阴凉。此外还有修建的很好的草坪，整洁的石子路，路两旁盛开的花朵，说明肯定有人负责打理这里，只是我从来没有见过他。我猜他就是是守墓人，可每次我去房间都是空的，看来他并不住在这里，也许他是一个比较忙的守墓人吧。如果有机会见到他，我就会建议他给这个墓地取个名字，把牌子立在入口，老是这里那里的指称我自己都感到有些腻烦。

那些不知道洛阳的人

发表于 2020-07-14 分类于 小说

她说起被骗往H城打工的事情时也没有表现得多么愤怒，更多的是以向过去未来撒娇使性的口气告诉我们那家工厂的老板让她们一帮女孩子连夜加班干活赶订单，而且吃的也不好，整天就是馒头大葱，最要命的是不能洗澡。说完这些她就给我们讲起了H城的牡丹，说那里有好多好多的牡丹，不光有大的牡丹园，普通人家也会种上几从。还说在那里交了一个男朋友，是个会修机器的小伙子，就是他用摩托车载着她去看牡丹花海。她觉得牡丹真的很好看，看牡丹的那天下着点小雨让她觉得很浪漫很有意境，以至于她讲述的时候嘴角都带上了一丝微笑，旁边另一个女孩子也含笑听着。服装店里黄黄的灯光从天花板上倾泻在她脸上，整张脸看起来是蜡黄色的，描的又细又黑的眉毛下面是带点血丝的眼睛，眼球上有几处黄色的斑点。她说她是九二年人，只看长相的话真的不像，可听她说了上面的一番话后又让我不得不信。当时我就站在她的斜身后，随口接道“洛阳的牡丹也很出名，不过洛阳也很破，不是什么好地方”。然后她就回过头来问我“洛阳是哪里，没听说过”，我彻底相信了她是九二年生人。
